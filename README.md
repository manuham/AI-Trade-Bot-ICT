# GBPJPY-AI-Trade-Analyst-Bot
Build a complete system with three components that work together: an MT5 Expert Advisor, a Python FastAPI server, and a Telegram bot. The system captures GBPJPY chart screenshots from MT5 at London open (08:00 CET) and NY open (14:30 CET), sends them to Claude's API for institutional-grade analysis, and delivers trade setups via Telegram.
